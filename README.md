# 🍕 Pizza Sales Analysis (SQL Project)

## 📌 Overview
This project analyzes pizza sales data using SQL to uncover business insights such as revenue trends, customer preferences, and top-performing products.

---

## 🎯 Objectives
- Calculate total revenue and orders
- Identify best-selling pizzas
- Analyze revenue by category
- Track sales trends over time
- Find top pizzas within each category

---

## 🛠 Tools Used
- SQL (Joins, Aggregations, CTEs, Window Functions)

---

## 📊 Key Insights
- Large pizzas generate the highest revenue
- A small number of pizzas dominate total sales
- Certain categories contribute more than others
- Sales trends show peak days and seasonal patterns

---

## 📁 Dataset
The dataset includes:
- Orders
- Order details
- Pizza types
- Pizza prices

---

## 🚀 Advanced SQL Concepts Used
- Common Table Expressions (CTE)
- Window Functions (RANK, Running Total)
- Subqueries
- Aggregations
