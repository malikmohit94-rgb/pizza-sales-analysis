# 🍕 Pizza Sales Analysis (SQL Project)

## 🧠 Business Problem
The goal of this project is to analyze pizza sales data to identify key revenue drivers, customer preferences, and opportunities to improve sales performance.

---

## 📌 Overview
This project analyzes pizza sales data using SQL to uncover business insights such as revenue trends, customer preferences, and top-performing products.

---

## 🎯 Objectives
- Calculate total revenue and orders
- Identify best-selling pizzas
- Analyze revenue by category
- Track sales trends over time
- Find top pizzas within each category

---

## 🛠 Tools Used
- SQL (Joins, Aggregations, CTEs, Window Functions)

---

## 🧾 Key SQL Queries
- Total Revenue Calculation
- Top 5 Best-Selling Pizzas
- Revenue by Category
- Daily & Monthly Revenue Trends
- Top 3 Pizzas per Category (Window Function)

---

## 📊 Key Insights
- Large pizzas generate the highest revenue
- A small number of pizzas dominate total sales
- Certain categories contribute more than others
- Sales trends show peak days and seasonal patterns

---

## 💡 Business Recommendations
- Promote large-size pizzas through combo offers to maximize revenue
- Focus marketing on top-performing pizza types
- Improve or re-strategize low-performing categories
- Use peak sales trends to optimize staffing and inventory

---

## 📁 Dataset
The dataset includes:
- Orders
- Order details
- Pizza types
- Pizza prices

---

## 🚀 Advanced SQL Concepts Used
- Common Table Expressions (CTE)
- Window Functions (RANK, Running Total)
- Subqueries
- Aggregations
